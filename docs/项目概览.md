# MindFlow 项目概览

## 项目简介

MindFlow 是一款极简风格的开源 Markdown 编辑器，致力于提供流畅的写作体验。

## 技术栈

- **前端框架**: React 18 + TypeScript
- **桌面端**: Tauri + Rust
- **Web端**: Vite + React
- **移动端**: Flutter (计划中)
- **编辑器**: CodeMirror 6
- **构建工具**: Turbo (Monorepo)
- **包管理**: npm

## 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0
- Rust (用于桌面端开发)

### 安装依赖

```bash
npm install
```

或使用设置脚本：

```bash
./scripts/setup.sh
```

### 开发命令

```bash
# 启动桌面端开发服务器
npm run desktop:dev

# 启动Web端开发服务器
npm run web:dev

# 构建所有包
npm run build

# 代码检查
npm run lint

# 代码格式化
npm run format
```

## 开发规范

### 提交信息规范

遵循约定式提交（Conventional Commits）：

- `feat`: 新功能
- `fix`: Bug 修复
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 代码重构
- `perf`: 性能优化
- `test`: 测试相关
- `chore`: 构建/工具链相关

示例：

```bash
git commit -m "feat(editor): add markdown syntax highlighting"
git commit -m "fix(file): resolve file save issue on Windows"
```

### 分支策略

- `main`: 主分支，用于生产环境
- `develop`: 开发分支
- `feature/*`: 功能开发分支
- `bugfix/*`: Bug 修复分支
- `hotfix/*`: 紧急修复分支

详细说明请参考 [Git工作流规范](./Git工作流规范.md)。
