## 系统特性

- 安全稳定支持
  - 纯本地使用,无需联网即可使用,源文件保存
  - 代码完全开源,不会上传用户信息，全方位保障用户隐私和数据安全
  - 支持语法高亮、黑暗模式、国际化、演示模式、单独编辑、ppt模式、导出文件、内部跳转、自动排版、图床、LaTeX、Mermaid、PlantUML、Markmap等功能
- 美观大气
  - 系统采用极简的设计风格,没有多余的东西，跟Typora一样
  - 操作简单，界面采用文件夹 + 文件列表 + 编辑器三栏设计，可收起，功能一览无遗
  - 中文默认使用FiraCode字体
- 高性能
  - 系统对多文件做了特别的性能优化,保障使用不卡顿
  - 相比当前市面上套壳Web的方式轻巧很多，更简洁
- 多平台支持
  - 支持Mac、Windows、Linux、iOS、Android等多平台
  - 支持Web版，可在浏览器中使用
  - 支持桌面版，可在桌面中使用
  - 支持移动版，可在手机中使用
- 操作简单
  - 操作过程很轻巧,无任何花里花哨的功能
  - 支持自动排版，所见即所得，编辑体验媲美原生代码编辑器
  - 系统内置众多快捷键，有效提供写作效率
  
## 功能列表

- Markdown写作
  - 支持标准Markdown语法，包括标题、列表、代码块、引用、表格等支持
  - 语法高亮与格式化（类似代码编辑器体验）
  - 支持本地文件夹 + Markdown文件管理，文档以纯文本形式存储
- 渲染与预览
  - 支持实时预览渲染Markdown内容，内容包括图表、公式、脑图等扩展内容（Mermaid、PlantUML、Markmap、LaTeX等）
  - 支持导出为PPT、PDF、图片，用于展示与分享
  - 支持演示模式（类似幻灯片／演示文档），用于写作、分享、演讲等场景
- 额外功能
  - 内置大量快捷键，可提升编辑效率
  - 支持浅色、深色主题、多种字体、自定义配置项
  - 三栏布局（文件夹、文档列表、编辑器 + 预览），符合用户使用习惯，便于文档管理与结构化写作
- 数据存储
- 文档本地保存，无云同步依赖，用户对数据完全掌控
- 特殊性能优化，确保系统轻量、启动迅速，响应迅速
